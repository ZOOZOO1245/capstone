배송안내

배송 기본 정보

  • 배송비: 3,000원
  • 무료배송: 5만원 이상 구매 시
  • 배송업체: CJ대한통운
  • 평균 배송기간: 1-3일 (주말/공휴일 제외)

배송 시간 안내

  • 오전 12시 이전 주문/결제 완료 → 당일 출고
  • 오전 12시 이후 주문/결제 완료 → 익일 출고
  • 출고 후 1-2일 이내 수령 (지역별 상이)

배송 조회 방법

  1. 마이페이지 접속
  2. '주문/배송 조회' 메뉴 선택
  3. 해당 주문건의 '배송조회' 버튼 클릭
  4. 실시간 배송현황 확인 가능

배송 특이사항

  • 도서/산간지역
    • 추가 배송비 3,000원
    • 배송기간 1-2일 추가 소요
  • 해외배송
    • 현재 해외배송은 지원하지 않음
    • 추후 해외배송 서비스 도입 예정
